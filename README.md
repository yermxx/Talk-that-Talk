# 🍓 프로젝트 명
Talk that Talk


## 🐸 프로젝트 설명
이 프로젝트는 **반응형 웹사이트 포트폴리오**입니다. 모바일, 데스크탑에서 최적화된 화면을 제공합니다. <br />
HTML, CSS 강의를 들은 뒤 그동안 공부했던 것을 복습하고, 응용하기 위해 만들었습니다. <br />
디자인과 코딩 모두 본인이 진행하였습니다! 😊
